# jquery.mangafilter

画像（IMG 要素）をマンガっぽくできる jQuery プラグインです。


## Usage

```
$('img').mangafilter({'maxWidth': 300, 'maxHeight': 300});
```

maxWidth, maxHeight を大きくするとかなり時間がかかります。
